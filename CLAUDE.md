This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Purpose

This is a marketplace for sharing Claude Code skills, workflows, and other resources for agent-based coding. It is hosted under the [CAIR](https://github.com/cair) GitHub organization and targets AI researchers who want to share and discover reusable Claude Code skills and agentic workflows.

## Context

- Hosted under the [CAIR](https://github.com/cair) GitHub organization (Center for Artificial Intelligence Research)
- The marketplace is meant to lower friction for AI researchers adopting Claude Code by making community-contributed skills discoverable and reusable
